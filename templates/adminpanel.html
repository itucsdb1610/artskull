User Operations

User operations includes editing users and editing admins information. Because of that it needs highest previlliege admin account. Even if you are an admin you cannot edit the users' usernames

List Users

List all users signed up ordered by their names. From this list-group you can access user's profile pages and edit their info if you have that previllige level.

Add Admin

Add admins from registered users to the database and give a previllige level to the added user. Admins will be added according to the usernames. Before adding any user as admin please check users' profile pages. Give right previllige levels to the users.

List Admins

List all admins from all previlliege levels. Remove one user from admin database or edit the previlliege level of each admin. Please be careful.

Play Operations

Play operations includes operations for three different databases. Firstly there should be a content. Then a stage should be added with wanted information. After adding content and stage, you can add play information that will hold provided content's play information with provided stage information.

Add Content

You can add new content to the database. All users registered to the system will be able to see this content and comment on this content. After user comments on a content, content will be showed on the users' followers' timelines. Contents will be meaningful if there is a play time added for them.

List Contents

List all contents on the database. You can access specific pages that will show content's information from this list. You can edit content's information from this list and also remove a content.

Add Stage

Add new stage to the database. Stages represent where plays will take place. Many plays can occur on same stage. Be careful to provide real information.

List Stages

List all stages available on database, edit stages if real values was changed or remove a stage from database.

Add Play

Add plays to the database. Content and stage data should added before adding play. Play data has to reference a stage and a content.

List Plays

List all plays available on the database. Add new plays, edit existing plays or remove a play entirely from database.
